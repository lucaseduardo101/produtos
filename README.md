# Desafio: Otimização do Endpoint de Produtos

## Objetivo:
Modificar o endpoint que retorna todos os produtos para eliminar duplicatas.
## Tarefas:
* Alterar o UseCase e Service para remover produtos duplicados antes de retorná-los.
* Implementar um algoritmo eficiente para remover duplicatas, otimizado para grandes volumes de dados.
Considerar produtos como duplicados se tiverem o mesmo nome e preço.

Restrições:
Usar apenas funcionalidades padrão do Kotlin, sem bibliotecas externas.
Manter o tempo de resposta do endpoint eficiente.

## Entrega:
* Código-fonte atualizado (UseCase e Service).
* Explicação do algoritmo escolhido e sua otimização.
